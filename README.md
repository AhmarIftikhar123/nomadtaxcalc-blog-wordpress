# Blocksy Child Theme - Modern WordPress Architecture

A high-performance WordPress child theme built for `nomadtaxcalc.com`. This project showcases a modern development workflow, integrating professional tooling into the WordPress ecosystem.

## 🚀 Key Features Meta-Built
- **Vite + Dart Sass**: Ultra-fast HMR and modern @use/@forward structure (7-1 pattern).
- **Custom Features**: 
  - **Reading Progress Bar**: Smooth, performant scroll tracking on single posts.
  - **Flexible TOC**: Configurable Table of Contents shortcode with depth control.
  - **Integrated Search & Mega Menu**: Seamlessly bridged Blocksy core with custom header logic.
- **Architectural Excellence**: PSR-4 Composer autoloading, Singleton patterns, and clean separation of concerns.

## 🛠️ Tech Stack
- **Languages**: PHP (7.4+), JavaScript (ES6+), SCSS (Dart Sass).
- **Core**: WordPress + Blocksy Parent Theme.
- **Tools**: Vite, Composer, NPM, Git.
- **AI Assisted**: Optimized using Google Antigravity & V0.dev.
