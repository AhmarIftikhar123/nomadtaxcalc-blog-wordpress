# Smart Home Mega Menu — Setup Guide

**HOW TO USE (no code editing needed after setup):**
─────────────────────────────────────────────────
1. Go to **Appearance → Menus** in your WordPress dashboard.
2. Add a top-level item (e.g. "Features") — this becomes the nav link.
3. Add child items under it — they AUTO-become the mega menu grid cards.
4. For each child item:
   - **Title**: The feature name (e.g., "Smart Lighting").
   - **Description**: The subtitle text. 
     *(Enable via Screen Options → Description at the top right of the Menus screen if you don't see the Description field).*
   - **Featured URL (Icon)**: Paste an image URL in the "URL" field prefixed with `img:` 
     *(Example: `img:https://yoursite.com/wp-content/uploads/icon.png`)*.
     OR upload to Media Library and paste the direct URL with the `img:` prefix.
   - **Default Icon**: Leave the URL blank or use `#` to show a default fallback icon.
   - **Normal Link**: If there is no `img:` prefix, the URL functions as a regular link.
5. Save the menu. Done. No code touching ever.
